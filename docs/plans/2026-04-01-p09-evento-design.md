# P09 — Evento: "DevForge 2026"

## Concepto
Conferencia tech/dev. Dark con glassmorphism, timeline animada, speakers carousel.

## Tipografía
- Headings: Geist Mono / JetBrains Mono fallback (monospace tech)
- Body: DM Sans (limpio)

## Paleta
- Fondo: `#09090f`, Surface: `rgba(255,255,255,0.03)`
- Text: `#e4e4e7`, Muted: `#71717a`
- Accent: `#a78bfa` (violeta) + `#38bdf8` (sky)
- Glass: bg `rgba(255,255,255,0.05)` border `rgba(255,255,255,0.08)`

## Secciones
1. Nav: DEVFORGE | Schedule, Speakers | Tickets
2. Hero: nombre monumental, fecha, CTA, glass stats card
3. Speakers: Swiper carousel glass cards
4. Schedule: timeline vertical con tabs Day 1/2/3
5. Tickets: 3 tiers glassmorphism
6. Sponsors: logos row
7. Footer

## Técnico
- GSAP + ScrollTrigger + Swiper via CDN
- Glassmorphism como elemento principal
- Archivo: evento-conferencia.html en raíz
